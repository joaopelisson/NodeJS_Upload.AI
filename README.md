## Getting Started

### Prerequisites

- NodeJS
- Yarn or NPM

### Installation


1. Clone the repository: 

```
git clone https://github.com/joaopelisson/NodeJS_Upload.AI.git
```

3. Install the necessary packages:
```
npm i
or
yarn install
```

4. Run the project:
````
npx prisma studio (view database)
npm run dev
````
